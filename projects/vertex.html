Vertex

Media Player Concept

In collaboration with Chris Wilczewski

What is it?

Vertex is a home media player built for simplicity without subtracting from the beautiful user interface. It minimizes the complications that come with proprietary software like iTunes and allows users to drag and drop movies in a manner comparable to a flash drive. Vertex can be controlled via a mobile app to forward the ongoing effort to eliminate the need for a television remote.

DESIGN

Vertex was designed with ease of use in mind. The goal was to ensure the user never became frustrated or confused when trying to find something to watch; the experience should be quick and painless from beginning to end. One approach in achieving this was the search layout. Inspiration was drawn from train station ticket booths--something that is familiar to a large percentage of future users. Upon input of text, the movies displayed on the screen change to reflect what has been entered and the alphabetized list along the bottom fades out as the search becomes more refined.

The interface for Vertex's home screen is minimalistic and only shows users the essentials. Upon launching, the user will be given the option of continuing where they left off with the most recent movie. By eliminating cumbersome information, Vertex offers a speedy, easy to use interface that allows users to be able to find what they are looking for exactly when they want it.

REMOTE

Part of the main experience when using Vertex is the Vertex Remote. This remote allows the user to perform all of the same functions as a normal remote and more. It features the standard Pause, Play, Rewind and Fast Forward, as well as performs searches and retrieves information for the movie currently being watched, such as Director, Cast, Year, Rating etc.

>

All the functions of Vertex Remote are controlled using intuitive gestures, Double-tap to pause or resume, Spin in a circle clockwise or counter clockwise to fast forward or rewind and swipe to the left and right to skip ahead or behind in the movie.
